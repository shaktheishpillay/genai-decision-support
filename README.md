# GenAI Decision Support System

Enterprise AI governance platform with human-in-the-loop approval workflows.

## Features

- **Policy-Based Risk Assessment**: Strict, Balanced, and Relaxed evaluation modes
- **Structured Decision Output**: Confidence scoring, risk flagging, and explanations
- **Human-in-the-Loop Governance**: Approve, reject, or request revisions
- **Audit Trail Logging**: Full compliance tracking for regulated industries

## Tech Stack

- Python
- Streamlit
- OpenAI API
- SQLite

## Local Setup
```bash
# Install dependencies
pip install -r requirements.txt

# Add your OpenAI API key to .env
OPENAI_API_KEY=your_key_here

# Run the app
streamlit run app.py
```
